Projects

Here are some of my notable projects that showcase my technical skills and expertise in AI, data science, and software development.

Enhancing Brain Tumour Segmentation in Multimodal MRI Scans

View Project on GitHub

During my Master's dissertation, I focused on developing sophisticated methods for the automated segmentation of brain tumors using multimodal MRI scans, tackling significant challenges in the field of medical image analysis.

My work emphasized the integration of cutting-edge techniques into convolutional neural network (CNN) architectures, such as attention mechanisms, multi-scale deep supervision, and multi-objective loss functions. Notably, incorporating Swin-Transformers into these architectures greatly enhanced the model's capacity to understand the global context within MRI scans. Utilizing the Brain Tumor Segmentation (BraTS) 2023 dataset, which includes various MRI types and expert annotations, I constructed three incremental 3D CNN architectures to explore different technique combinations, with the deep supervision CNN model achieving an impressive mean Dice score of 0.795.

Dialogue Insights Generator: An Innovative NLP Solution

View Project on GitHub

Developed an innovative web application using Flask and Python, leveraging state-of-the-art NLP techniques, Generative AI models (GPT-3.5-turbo), proprietary diarization pipelines (Deepgram, WhisperX) from Huggingface, and cloud deployment (using Docker and GCP) to provide comprehensive insights from conversational data.

The solution transcribes audio dialogues, analyses conversation text, and generates personalized sentiment and psychological profiles for each speaker, while also assessing speaker attitudes and recommending strategies for future interactions.

Fortune 1000 CEO Diversity and Performance Dashboard

View Dashboard on Tableau Public

Developed an interactive Tableau dashboard to analyze CEO gender diversity and financial performance trends of Fortune 1000 companies from 2019-2021. This comprehensive visualization tool identifies insights into gender disparities across sectors and correlations between leadership demographics and business growth.

The dashboard provides valuable insights for investors, business leaders, and diversity advocates, highlighting the business case for diversity in corporate leadership.

Scholarship Prediction and Early Identification of At-Risk Students

View Project on GitHub

Designed and implemented ML and DL models to predict student dropout and academic success using demographic, socioeconomic, and academic data. Utilized Python and R with libraries like Pandas, Scikit-learn, and Keras to build and train models, performing extensive preprocessing including imputation, outlier removal, and class balancing.

Conducted exploratory data analysis to uncover feature relationships and built various models including Logistic Regression, KNN, Random Forest, Decision Tree, SVM, and CNNs. Models were evaluated using 5-fold stratified cross-validation, with the CNN achieving the highest accuracy of 76% for predicting scholarship awards.

Missing Money Matters - Data Analysis Project using SQL and NoSQL

View Project on GitHub

The "Missing Money Matters" project is an engaging SQL and NoSQL training exercise where I played the role of a data analyst for WSDA Music. Tasked with investigating financial discrepancies in the company's records, I utilized SQLite and MongoDB to analyze and manipulate data.

The project demonstrates proficiency in both relational and non-relational database systems, showcasing the ability to extract meaningful insights from complex financial data across different database paradigms.
